# calculadora :notebook_with_decorative_cover:

Um app calculadora padrão desenvolvido usando _react_ e _styled-components_.
Possui as seguintes operações:

* Adição (+)
* Subtração (-)
* Multiplicação (*)
* Divsão (/)
* Porcentagem (%)
* Potência base 10 (10<sup>x</sup>)
* Raiz quadrada &radic;x
* x quadrado (x&sup2;)
* Valor inverso (1/x)
* Sinal (-)
